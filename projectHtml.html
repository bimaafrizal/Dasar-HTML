Pengenalan HTML


1. Pengertian

HTML adalah singkatan dari Hyper Text Markup Language. HTML adalah bahasa markup standar untuk membuat halaman Web.
Markup language adalah suatu set markup tag yang bertujuan untuk mendeskripsikan sebuah halaman. HTML menggambarkan struktur halaman Web.
Sebagai pembuat halaman web, html tidak bekerja sendirian Didalam website terdapat tiga layer yang bekerja

2. Layer Web

Dalam Web terdapat tiga layer yaitu :

  • HTML

    html berfungsi untuk menambah elemen dan membentuk struktur konten

  • CSS

    CSS berkaitan dengan styling seperti background, warna, layout, spacing dan animasi

  • JavaScript

    JavaScript memungkinkan anda menambahkan fungsionalitas yang dinamis seperti slider, pop-up, dan galeri
