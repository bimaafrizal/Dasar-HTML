1. HTML

HTML adalah singkatan dari Hyper Text Markup Language. HTML adalah bahasa markup standar untuk membuat halaman Web.
Markup language adalah suatu set markup tag yang bertujuan untuk mendeskripsikan sebuah halaman. HTML menggambarkan struktur halaman Web.
Sebagai pembuat halaman web, html tidak bekerja sendirian Didalam website terdapat tiga layer yang bekerja


Pengertian TAG dalam HTML

Tag HTML memiliki dua tipe utama: block-level dan inline tags.

  • Elemen block-level

    Elemen block-level memakai semua space yang tersedia dan selalu membuat line baru di dalam dokumen. Contoh dari tag block adalah heading dan paragraf.

  • Elemen inline

    Elemen inline hanya memakai space sesuai dengan kebutuhannya dan tidak membuat line baru di halaman. Biasanya elemen ini akan memformat isi konten dari elemen block-level. Contoh dari tag inline adalah link dan emphasized strings.


2. CSS

Cascading Style Sheets atau lebih dikenal dengan CSS adalah bahasa pemrograman desain yang berguna untuk menyederhanakan proses pembuatan website.
CSS merupakan bahasa pemrograman yang dipakai untuk mendesain halaman depan atau tampilan website (front end). CSS menangani tampilan dan ‘rasa’ dari halaman website.
